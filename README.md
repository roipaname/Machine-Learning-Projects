

# AI Projects

This repository contains multiple machine learning projects demonstrating a wide range of skills in data ingestion, processing, modeling, and deployment. Each project is self-contained, focusing on practical applications and real-world scenarios.

---

### House Price Prediction (API + DB)

* **What was done:** Pulled housing data from a public API, stored in PostgreSQL, performed feature engineering, trained regression models.
* **Key tech:** Python, requests, pandas, PostgreSQL, API handling, feature engineering.

---

### News Article Scraper & Topic Classifier

* **What was done:** Scraped news websites, normalized text, extracted features using TF-IDF, classified articles into topics, handled duplicate articles.
* **Key tech:** BeautifulSoup, SQLite, scikit-learn, text preprocessing, TF-IDF, classification.

---

### Twitter/X Sentiment Analyzer

* **What was done:** Fetched tweets via API, processed them for sentiment analysis, handled rate limits, stored results in MongoDB.
